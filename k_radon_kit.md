# K-Radon kit - 마스터 가이드

본 문서는 **K-Radon kit**의 프로젝트 소개, 시스템 구조, 페이지별 상세 구현 내용 및 전체 개발 이력을 통합한 마스터 가이드입니다.

---

## 1. 프로젝트 개요

### 1.1 프로젝트 정의
- **타이틀**: K-Radon kit
- **목적**: 측정된 라돈 데이터를 그래프를 통해서 확인하고 분석하도록 구축된 웹 기반 통합 포털 시스템입니다.

### 1.2 핵심 기능
- **회원 가입 및 로그인**: 이메일,패스워드,이름,회사명,전화번호로 간단하게 회원 가입을 통해 로그인 후 관리자 권한을 획득하여 시스템을 이용할 수 있습니다.
- **라돈 측정기 연결 및 데이터 확인**: 시리얼(UART) 통신을 통해서 라돈 측정기에서 측정된 라돈 데이터를 로드하여 그래프를 통해서 확인하고 분석하도록 구축된 웹 기반 통합 포털 시스템입니다.
- **과금 전용 기능**: 사용한 데이터 양에 따라 과금되는 기능을 제공합니다. 예상 비용을 보여주고 세금 계산서 발행을 통하여 과금되는 기능을 제공합니다.
- **사용자 정보**: 이메일을 제외한 패스워드,이름,회사명,전화번호를 수정할 수 있습니다.
- **화면 구성**: 상단 메뉴에 '장비', '과금', '사용자 정보'로 구성되어 있습니다.

### 1.3 기술 스택
- **Frontend**: React Native
- **Backend**: REST API를 통하여 데이터를 주고 받습니다.
